# ZORBS 🔵 Stream Racer

Interactive browser racing game for Twitch/Kick streamers.

## Local Dev
```bash
npm install && npm run dev
# Open http://localhost:3000
```

## Deploy (one time)
```bash
npm i -g vercel
vercel --prod
```

## Chat Commands
| Command | Who | What |
|---------|-----|------|
| `!play` | Anyone | Join race |
| `!boost` | Anyone | Speed boost (subs 2x) |

## Streamer Panel
Set optional race duration (e.g. `3m`, `90s`) then click START RACE.
No duration = race ends when someone crosses the finish line.
